## 一、关键词研究（Keyword Research）

关键词研究是一切 SEO 工作的起点。没有精准的关键词定向，后续所有努力都可能方向偏差。

### 1.1 工具使用能力

- [ ] 熟练使用 **Google Keyword Planner**，能读懂搜索量区间与竞争程度指标
- [ ] 掌握 **Ahrefs / Semrush / Moz** 等专业工具的关键词探索、难度评估（KD）和流量预测功能
- [ ] 会使用 **Google Search Console** 挖掘已有排名词汇的潜力词
- [ ] 了解 **Answer The Public、AlsoAsked、People Also Ask** 等问答型关键词挖掘工具
- [ ] 能借助 **Google Trends** 判断关键词的季节性与趋势走向

### 1.2 关键词分析能力

- [ ] 能区分 **短尾词（Head Terms）**、**中尾词（Mid-tail）** 与 **长尾词（Long-tail）** 的战略价值
- [ ] 能评估关键词的 **商业价值**：信息型 / 导航型 / 交易型 / 商业调研型（TOFU / MOFU / BOFU）
- [ ] 掌握关键词 **竞争难度（KD）** 分析，结合域名权重制定可实现的目标
- [ ] 能识别 **关键词蚕食（Keyword Cannibalization）** 问题，避免内部页面互相竞争
- [ ] 具备 **本地化关键词** 研究能力（城市 + 行业词、地区方言词等）

### 1.3 关键词规划能力

- [ ] 能构建完整的 **关键词矩阵**（按主题分组、按页面类型映射）
- [ ] 能制定合理的 **关键词优先级排序策略**（低竞争 × 高价值优先入局）
- [ ] 会做竞争对手关键词的 **Gap Analysis（差距分析）**，找到竞品排名但自身缺失的词

## 二、页面内优化（On-Page SEO）

On-Page SEO 是确保每一个页面都能被搜索引擎正确理解和用户喜爱的基础工作。

### 2.1 元数据优化

- [ ] 能撰写 **点击率（CTR）驱动型 Title Tag**，控制在 55–60 字符内，包含核心关键词
- [ ] 能撰写吸引用户点击的 **Meta Description**，字数控制在 150–160 字符，包含 CTA
- [ ] 了解 **robots meta 标签**（noindex、nofollow）的正确使用场景
- [ ] 能配置 **Open Graph / Twitter Card 标签**，优化社交分享预览效果
- [ ] 掌握 **Canonical 标签** 的设置方法，防止重复内容问题

### 2.2 内容结构优化

- [ ] 掌握 **H1–H6 标题层级** 的语义化使用原则（每页仅一个 H1）
- [ ] 能合理布局 **关键词密度与 LSI 语义相关词**，避免堆砌
- [ ] 熟悉 **Featured Snippet 优化** 技巧（段落型、列表型、表格型 Snippet 的内容结构）
- [ ] 能优化 **图片 Alt 属性**，兼顾无障碍访问与关键词植入
- [ ] 掌握 **内部链接策略**：Pillar Page 与 Cluster Content 的链接架构

### 2.3 URL 与结构优化

- [ ] 能设计 **简洁语义化的 URL 结构**，避免冗余参数
- [ ] 了解 **面包屑导航（Breadcrumb）** 对 SEO 的结构性价值
- [ ] 掌握 **301 / 302 重定向** 的正确使用场景，避免链接权重损失

## 三、技术 SEO（Technical SEO）

技术 SEO 是整个网站的"骨架"，决定了搜索引擎能否顺畅地爬取、渲染和索引你的内容。

### 3.1 爬取与索引控制

- [ ] 能读懂并配置 **robots.txt** 文件，正确屏蔽非目标页面
- [ ] 熟悉 **XML Sitemap** 的生成与提交，知道哪些 URL 不应出现在 Sitemap 中
- [ ] 能使用 **Google Search Console 的覆盖率报告** 发现索引问题（已排除、已爬取但未索引等）
- [ ] 掌握 **爬取预算（Crawl Budget）** 概念，能通过日志文件分析 Googlebot 的爬取行为
- [ ] 能处理 **孤立页面（Orphan Pages）**：找出无内链指向的页面并补全链接

### 3.2 网站速度优化

- [ ] 熟悉 **Core Web Vitals（LCP / FID / CLS）** 指标含义与优化方向
- [ ] 能使用 **PageSpeed Insights、Lighthouse、GTmetrix** 进行性能诊断
- [ ] 掌握 **图片压缩与 WebP 格式转换**、懒加载（Lazy Load）实现方法
- [ ] 了解 **CDN 配置、浏览器缓存策略、Gzip/Brotli 压缩** 等服务器端优化手段
- [ ] 能评估并优化 **第三方脚本（如 GTM、Chat Widget）** 对加载速度的影响

### 3.3 移动端与安全

- [ ] 理解 **Mobile-First Indexing** 的本质，能审查移动端页面体验
- [ ] 掌握 **响应式设计（Responsive Design）** 与 **动态服务（Dynamic Serving）** 的区别
- [ ] 能配置 **HTTPS（SSL 证书）**，处理 HTTP → HTTPS 迁移中的混合内容问题

### 3.4 结构化数据（Schema Markup）

- [ ] 掌握 **JSON-LD 格式** 的结构化数据编写，优先于 Microdata
- [ ] 能实现常用 Schema 类型：Article、Product、FAQ、HowTo、BreadcrumbList、Organization、LocalBusiness
- [ ] 能使用 **Google Rich Results Test** 验证标记有效性
- [ ] 了解 **结构化数据与 Featured Snippet / Rich Result 的关联关系**

### 3.5 国际化 SEO

- [ ] 掌握 **hreflang 标签** 的正确配置（自引用 + 双向互指原则）
- [ ] 能制定多语言/多地区网站的 **URL 结构方案**（ccTLD / 子域名 / 子目录）
- [ ] 了解 **Geotargeting 设置**（在 Search Console 中为子目录指定目标国家）

## 四、内容策略（Content Strategy）

好的内容策略不是"写文章"，而是系统性地规划什么内容、为谁创作、何时发布、如何组织。

### 4.1 内容规划能力

- [ ] 能构建 **主题聚类模型（Topic Cluster Model）**：一个 Pillar Page 对应多个 Cluster Pages
- [ ] 能制定 **内容日历（Editorial Calendar）**，结合季节性关键词趋势安排发布节奏
- [ ] 能做 **内容审计（Content Audit）**：识别高潜力旧文、低价值内容与重复内容
- [ ] 掌握 **竞争内容分析**（SERP 前 10 内容结构对比，找出超越机会）
- [ ] 能识别 **内容缺口（Content Gap）**，将用户旅程中的空白页面补全

### 4.2 内容创作能力

- [ ] 能撰写符合 **E-E-A-T 标准**（经验、专业度、权威性、可信度）的高质量内容
- [ ] 掌握 **内容深度优化**：覆盖同一主题的关联子话题，提升主题相关性信号
- [ ] 熟悉 **多媒体内容融合**：适时嵌入图片、视频、信息图、表格提升停留时长
- [ ] 能进行 **内容更新（Content Refresh）**：找出排名下滑的旧文并系统性改进
- [ ] 了解 **AI 辅助写作工具**（如 Jasper、Surfer SEO）的合理使用边界

### 4.3 用户意图匹配

- [ ] 能准确判断关键词的 **搜索意图类型**（信息型 / 导航型 / 交易型 / 商业调研型）
- [ ] 能根据意图类型选择 **最合适的内容格式**（博客 / 落地页 / 比较页 / 工具页）
- [ ] 能分析 **SERP 特征（SERP Features）**（视频、图片、购物卡等）推断 Google 对该词意图的理解

## 五、外链建设（Link Building / Off-Page SEO）

外链是 Google 算法中历史最悠久的权威性信号之一，也是最难规模化复制的竞争壁垒。

### 5.1 链接评估能力

- [ ] 能评估外链质量的核心指标：**Domain Rating（DR）、Domain Authority（DA）、Trust Flow**
- [ ] 掌握 **链接相关性判断**：话题相关 > 单纯高权重
- [ ] 能识别 **有毒外链（Toxic Links）**，知道何时使用 **Disavow Tool**
- [ ] 熟悉 **竞争对手外链分析**：用 Ahrefs / Semrush 挖掘可复制的外链来源

### 5.2 链接获取策略

- [ ] 掌握 **Guest Posting（客座博客）** 的选站标准与内容质量要求
- [ ] 熟悉 **Broken Link Building（失效链接替换）** 的完整操作流程
- [ ] 了解 **Digital PR / 数字公关** 策略，通过新闻资源、数据报告获取媒体链接
- [ ] 能执行 **Skyscraper Technique（摩天楼技术）**：创作超越竞品的内容并主动推广
- [ ] 掌握 **Niche Edit / Link Insertion** 的操作方法与筛选标准
- [ ] 了解 **HARO（Help A Reporter Out）** 等媒体资源对接平台的使用方法

### 5.3 链接合规意识

- [ ] 熟悉 **Google 链接垃圾政策（Link Spam Policy）**，避免触碰黑帽链接手段
- [ ] 了解 **付费链接、互换链接、PBN（私人博客网络）** 的风险与 Google 识别机制
- [ ] 能判断自然链接增长曲线，识别异常速率导致的 **Penguin 算法风险**

## 六、数据分析（SEO Analytics）

数据能力决定了 SEO 工作者能否从"执行层"进阶到"策略层"。

### 6.1 工具使用能力

- [ ] 深度掌握 **Google Search Console**：搜索效果、覆盖率、Core Web Vitals、链接报告
- [ ] 熟练使用 **Google Analytics 4（GA4）**：自然流量分析、转化路径、页面参与度指标
- [ ] 能使用 **Screaming Frog / Sitebulb** 进行站内爬取审查
- [ ] 熟悉 **Ahrefs / Semrush** 的排名追踪、流量预测与竞争对手分析模块
- [ ] 能使用 **Looker Studio（原 Google Data Studio）** 构建 SEO 汇报仪表盘

### 6.2 数据解读能力

- [ ] 能识别 **排名波动的根本原因**：算法更新 vs. 内容质量 vs. 技术问题 vs. 外链变化
- [ ] 掌握 **CTR 优化分析**：找出高展示低点击的页面，优化 Title/Description
- [ ] 能区分 **直接流量、自然流量、推荐流量** 的数据混淆问题（如 Dark Traffic）
- [ ] 了解 **归因模型（Attribution Model）**，避免将 SEO 价值低估
- [ ] 能基于数据进行 **A/B 测试**：标题测试、内容结构测试、CTA 测试

### 6.3 汇报与沟通能力

- [ ] 能将 SEO 数据转化为 **业务语言**（流量价值、潜在转化、ROI 估算）
- [ ] 能制作清晰的 **月报 / 季报**，包含核心指标趋势、阶段成果与下期计划
- [ ] 能向非技术决策者解释 **为什么 SEO 是长期投资**，设定合理的预期

## 七、用户意图与体验（User Intent & UX）

Google 的核心使命是"连接用户与最有用的信息"，因此用户体验信号日益成为排名因素。

### 7.1 搜索意图深度理解

- [ ] 能分析 **SERP 前 10 结果的内容形态**，反推 Google 对该词的意图判断
- [ ] 能识别 **意图转变（Intent Shift）**：同一关键词在不同时期意图的演变
- [ ] 能区分 **显性意图（Explicit Intent）** 与 **隐性意图（Implicit Intent）**，在内容中同时覆盖

### 7.2 用户体验优化

- [ ] 理解 **跳出率（Bounce Rate）、页面停留时长、滚动深度** 对 SEO 的间接影响
- [ ] 掌握 **CTA 设计与内部链接优化**，引导用户深度浏览降低跳出
- [ ] 了解 **无障碍设计（Web Accessibility）** 对 SEO 的正面价值（Alt 文本、字幕、语义化 HTML）

---

## 八、主题权威与垂直领域建设（Topical Authority）

主题权威是现代 SEO 中最难复制、但一旦建立就极具壁垒价值的竞争优势。

- [ ] 能围绕核心业务构建 **完整的主题覆盖地图**（Topic Map），确保没有重要子话题缺失
- [ ] 理解 **主题相关性（Topical Relevance）** 与 **域名权重（Domain Authority）** 的协同关系
- [ ] 能执行 **Topical Cluster 策略**：通过系统性内容覆盖让 Google 将站点识别为该领域专家
- [ ] 熟悉 **E-E-A-T 建设路径**：作者简介、外部媒体提及、行业认可、真实案例背书
- [ ] 能识别 **YMYL（Your Money Your Life）** 类页面的额外审核标准，针对性提升可信度

---

## 九、本地 SEO（Local SEO）

对于本地服务类业务，本地 SEO 往往是 ROI 最高的增长渠道。

- [ ] 熟练操作 **Google Business Profile（GBP）**：完整填写、分类选择、关键词植入描述
- [ ] 了解 **本地引用（Local Citations）** 的一致性要求（NAP：Name、Address、Phone）
- [ ] 能执行 **本地评价策略**：通过邮件/短信引导满意客户留下真实好评
- [ ] 掌握 **本地落地页优化**：区域词 + 服务词的组合，针对多城市业务的页面矩阵设计
- [ ] 了解 **Local Pack（地图三件套）** 的排名影响因素

---

## 十、电商 SEO（E-commerce SEO）

电商 SEO 具有其独特的技术复杂度与内容规模化挑战。

- [ ] 掌握 **商品页面（PDP）** 的 SEO 优化：唯一性描述、评论 UGC、结构化数据
- [ ] 能处理 **分页（Pagination）** 的 SEO 问题（rel=next/prev 已废弃，推荐加载更多 + Canonical）
- [ ] 了解 **分面导航（Faceted Navigation）** 的爬取预算管理与 URL 参数过滤
- [ ] 掌握 **季节性内容策略**：大促前的内容提前布局（至少提前 3 个月）
- [ ] 熟悉 **Google Shopping / Merchant Center** 与自然搜索的协同逻辑

---

## 十一、SEO 项目管理与协作能力

SEO 不是孤立的个人工作，需要与开发、内容、产品、市场团队高效协作。

- [ ] 能撰写清晰的 **SEO 技术需求文档（Tech Brief）**，让开发工程师准确理解实现要求
- [ ] 掌握基础的 **HTML / CSS / JavaScript** 知识，能独立完成简单的前端改动或提出精准的修改建议
- [ ] 了解 **CMS 系统**（WordPress、Shopify、Webflow 等）的 SEO 配置与插件管理
- [ ] 能进行 **SEO 优先级排序**，在资源有限时判断哪些工作影响最大
- [ ] 具备 **跨部门沟通能力**，能在业务目标框架下推动 SEO 建议落地

---

## 十二、持续学习与行业洞察

SEO 是一个持续演进的领域，2–3 年前的最佳实践今天可能已经过时。

- [ ] 定期阅读 **Google 官方博客、Search Central Blog**，追踪算法更新公告
- [ ] 跟进 **Search Engine Journal、Search Engine Land、Moz Blog、Ahrefs Blog** 等权威媒体
- [ ] 了解 **AI 搜索（AI Overview / SGE）** 对传统 SEO 的影响与应对策略
- [ ] 能快速分析 **Core Algorithm Update、Helpful Content Update、Spam Update** 等重大更新对自身站点的影响
- [ ] 积极参与 **SEO 社区**（Twitter/X SEO 圈、Reddit r/SEO、国内 SEO 论坛），获取一手实战经验

## 写在最后

SEO 的本质，是通过持续地为用户创造真实价值，赢得搜索引擎的信任。**没有哪一个台阶是可以跳过的**——关键词研究决定了努力的方向，技术 SEO 确保了内容能被看见，内容策略决定了是否值得被推荐，外链建设证明了你在行业中的地位，数据分析让迭代有据可依。

真正的 SEO 专家，是能在这套系统中找到杠杆点、高效配置资源、并持续复利增长的人。

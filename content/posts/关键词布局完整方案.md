## 一、网站结构分析与关键词分层策略

### 典型的管件产品网站结构

```
首页
│
├─ 产品分类页（10-20个）
│  ├─ Stainless Steel Pipe Fittings (主分类)
│  ├─ Carbon Steel Pipe Fittings
│  ├─ Brass Pipe Fittings
│  └─ PVC Pipe Fittings
│
├─ 产品子分类页（50-100个）
│  ├─ Stainless Steel Elbows
│  ├─ Stainless Steel Tees
│  ├─ Stainless Steel Reducers
│  └─ Stainless Steel Flanges
│
└─ 具体产品页（400-500个）
   ├─ 304 Stainless Steel 90 Degree Elbow 1/2"
   ├─ 304 Stainless Steel 90 Degree Elbow 3/4"
   └─ 316 Stainless Steel Tee 1"
```

### 关键词分层原则

**第1层 - 首页：**
- 品牌词 + 行业核心词
- 例：`[Brand Name] - Pipe Fittings Manufacturer & Supplier`

**第2层 - 主分类页（高搜索量，高竞争）：**
- `Stainless Steel Pipe Fittings`
- `Carbon Steel Pipe Fittings`

**第3层 - 子分类页（中等搜索量）：**
- `Stainless Steel Elbow Fittings`
- `Stainless Steel Tee Fittings`

**第4层 - 产品页（长尾词）：**
- `304 Stainless Steel 90 Degree Elbow`
- `1/2 inch stainless steel elbow`

## 二、使用 SEMrush 进行关键词研究与筛选

### Step 1: 竞争对手分析（找到关键词机会）

**操作路径：**
1. 打开 SEMrush → `Competitive Research` → `Organic Research`
2. 输入3-5个竞争对手域名
3. 点击 `Positions` 查看他们排名的关键词

**需要导出的数据：**
```
Domain Comparison
├─ 竞争对手A排名但你没排名的词（机会词）
├─ 搜索量 > 100的词
└─ KD（关键词难度）< 60的词
```

**实操示例：**
```
竞争对手：pipefittingssupplier.com
导出筛选条件：
- Search Volume: 100-10,000
- Keyword Difficulty: 0-50
- Position: 1-20 (他们的排名)
- Your domain position: 不存在（你还没排名）
```

**SEMrush 截图位置提示：**
- 点击 `Advanced Filters`
- 添加过滤器：`Include keywords` → `containing` → 输入 `pipe fittings`
- 导出为 Excel

### Step 2: 关键词魔法工具（Keyword Magic Tool）

**操作步骤：**

1. **进入工具：**
   `Keyword Analytics` → `Keyword Magic Tool`

2. **输入种子词：**
   ```
   stainless steel pipe fittings
   ```

3. **设置筛选条件：**
   ```
   Advanced Filters:
   ├─ Search Volume: 50-5000（避免太高竞争或无流量词）
   ├─ Keyword Difficulty: 0-40（可实现的难度）
   ├─ Word Count: 3-6（长尾词）
   └─ Intent: Commercial/Transactional（购买意图）
   ```

4. **使用左侧分组功能：**
   - 点击 `Broad Match`、`Phrase Match`、`Exact Match`、`Related`
   - 查看 `Questions`（问题类关键词，适合博客/FAQ）
   - 点击词组分组（如：`elbow`、`tee`、`coupling`）

5. **导出关键词列表：**
   - 勾选相关词
   - 点击 `Add to Keyword Manager` 或直接 `Export`

### Step 3: 关键词分组与页面映射

**在 Excel 中操作（或使用 SEMrush Keyword Manager）：**

### Step 4: 识别关键词蚕食问题

**使用 Position Tracking：**

1. `Position Tracking` → 添加你的域名
2. 添加你筛选好的关键词列表
3. 等待1-2天获取数据
4. 点击 `Cannibalization` 标签

**会显示：**
```
关键词: stainless steel elbow
├─ URL 1: /products/elbow-123/ (排名45)
├─ URL 2: /stainless-steel-elbows/ (排名38)
└─ URL 3: /blog/elbow-guide/ (排名52)

⚠️ 同一关键词有3个页面竞争 - 需要整合！
```

**解决方案：**
- 确定一个主页面
- 其他页面改为优化长尾变体
- 或者合并/301重定向

### Step 5: 内容差距分析

**操作：**
1. `Gap Analysis` → `Keyword Gap`
2. 输入你的域名和3个竞争对手
3. 查看 `Missing` 和 `Weak` 列

**会得到：**
- 竞争对手都排名但你缺失的词
- 你排名很低的词（优化机会）

**问题：** 你有50个相似的 elbow 产品页，都想优化 "stainless steel elbow"

**解决方案：**

1. **创建聚合页：** `/stainless-steel-elbows/`

   - 每个产品页：`304 stainless steel 90 degree elbow 1/2 inch NPT threaded`
   - Title 包含所有规格参数
   - 避免与聚合页竞争

## 七、避免的常见错误

❌ **错误1：所有产品页都优化相同关键词**
```
50个产品页都优化 "stainless steel elbow" → 关键词蚕食
```
✅ **正确做法：**
```
聚合页优化 "stainless steel elbow"
产品页优化 "304 ss elbow 1/2 inch" (超长尾)
```

❌ **错误2：完全依赖自动生成TDK**
✅ **正确做法：** 优先优化前100个重要页面，其余用模板但定期审查

❌ **错误3：忽略搜索意图**
```
"stainless steel pipe fittings suppliers" → 信息/列表页
"buy stainless steel elbow" → 产品页
```

## 二、是否需要聚合页？判断标准

### 需要创建聚合页的情况（✅ 你的情况）

**判断依据：**
1. ✅ 该二级类目下有 **15个以上** 产品
2. ✅ 类目关键词有 **独立的搜索量**（如 "stainless steel elbows" 月搜索500+）
3. ✅ 产品之间有明显的 **参数差异**（尺寸、材质、角度）
4. ✅ 用户需要 **对比选择**（不是直接知道型号）

**你的情况：**
```
二级类目：Elbows
├─ 产品数量：50个 ✅
├─ 关键词搜索量：stainless steel elbows (月搜索800) ✅
├─ 参数差异：材质、尺寸、角度、连接方式 ✅
└─ 用户行为：需要筛选和对比 ✅

结论：强烈建议创建聚合页！
```

### 不需要聚合页的情况（参考）

```
二级类目：Special Custom Fittings
```

```
1. 让聚合页优化主词
   └─ 50个产品页不要竞争同一个词

2. 产品页优化超长尾
   ├─ 加入所有规格参数到Title
   ├─ 每个页面独特的关键词组合
   └─ 示例：
      产品1: "304 Stainless Steel 90 Degree Elbow 1/2 Inch NPT Thread"
      产品2: "304 Stainless Steel 90 Degree Elbow 3/4 Inch Socket Weld"
      产品3: "316 Stainless Steel 90 Degree Elbow 1/2 Inch Butt Weld"

3. 内容差异化
   ├─ 技术参数表（每个产品不同）
   ├─ 应用场景（根据尺寸不同而不同）
   └─ FAQ（针对具体规格的问题）
```

### Q2: 没时间写800-1200词的聚合页内容？

**解决方案：**
```
优先级调整：

方案A (推荐):
├─ 先创建10个最重要的聚合页（搜索量最高）
├─ 每个1200词高质量内容
└─ 剩余10个暂时用600词，后续补充

方案B (折中):
├─ 所有聚合页先创建600-800词
├─ 包含核心的选购指南和FAQ
├─ 每月选3-5个扩展到1200词
└─ 3个月内全部达标

方案C (外包):
├─ 外包内容撰写（Fiverr/Upwork）
├─ 提供详细Brief和关键词
├─ 自己审核和调整
└─ 成本: $50-100/篇
```

### Q3: 聚合页和产品页如何避免内容重复?

**内容区分策略：**

```
聚合页 (/stainless-steel-elbows/):
├─ 宏观视角：什么是elbow，为什么需要
├─ 横向对比：不同材质/角度/尺寸的对比
├─ 选购指南：如何选择合适的elbow
├─ 行业应用：elbow在不同行业的应用
└─ 通用FAQ：关于elbow的一般问题

产品页 (/304-ss-elbow-90-1-2-npt/):
├─ 微观视角：这个具体产品的特点
├─ 具体参数：精确的技术规格
├─ 使用说明：这个产品怎么用
├─ 兼容性：配套哪些产品
└─ 特定FAQ：关于这个规格的具体问题

内容几乎0重复！
```

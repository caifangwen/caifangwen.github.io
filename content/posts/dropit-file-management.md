## 什么是 DropIt？

[DropIt](http://www.dropitproject.com/) 是一款免费、开源的 Windows 文件自动化整理工具。你只需将文件或文件夹**拖放**到 DropIt 的悬浮图标上，它便会根据你预先设置的规则，自动完成移动、复制、压缩、重命名、删除等操作，极大地减少手动整理文件的时间。

---

## 安装与启动

1. 前往 [DropIt 官网](http://www.dropitproject.com/) 下载最新版本（支持安装版与便携版）。
2. 安装完成后，桌面会出现一个**可拖动的悬浮图标**（默认为箭头形状）。
3. 右键图标可进入设置菜单；双击图标可打开主界面。

> **提示**：便携版无需安装，适合在 U 盘中携带使用。

---

## 核心概念

在正式使用前，先理解三个核心概念：

| 概念 | 说明 |
|------|------|
| **规则（Rule）** | 定义"什么文件"触发"什么操作"的匹配条件 |
| **配置文件（Profile）** | 一组规则的集合，可针对不同场景切换 |
| **关联文件夹（Association）** | 指定监控目录，实现全自动整理，无需手动拖放 |

---

## 创建整理规则

### 第一步：打开规则编辑器

右键悬浮图标 → 选择 **"关联和规则（Associations & Rules）"**，进入规则管理界面。

### 第二步：新建规则

点击左下角 **"+"** 按钮，填写以下信息：

#### 匹配条件（过滤器）

支持多种匹配方式：

```
*.jpg;*.png;*.gif        # 按扩展名匹配（多个用分号分隔）
报告_*                    # 按文件名前缀匹配（支持通配符）
*_2026*                  # 匹配文件名中包含 _2026 的文件
```

还可以设置**附加条件**，例如：
- 文件大小范围（如大于 10MB）
- 修改日期范围
- 文件夹层级深度

#### 执行动作

DropIt 支持 14 种以上的操作，常用操作如下：

| 动作 | 说明 |
|------|------|
| **移动（Move）** | 将文件移动到指定目录 |
| **复制（Copy）** | 复制文件到目标目录，保留原文件 |
| **压缩（Compress）** | 打包压缩为 ZIP/7Z 等格式 |
| **重命名（Rename）** | 按规则批量重命名 |
| **删除（Delete）** | 删除匹配的文件（谨慎使用） |
| **打开（Open with）** | 用指定程序打开文件 |
| **忽略（Ignore）** | 跳过该文件，不做任何处理 |

#### 目标路径设置

路径支持**动态变量**，例如：

```
D:\Documents\[year]\[month]\     # 按年/月自动分类
D:\Photos\[exif:year]\            # 按照片拍摄年份分类
D:\Downloads\[extension]\         # 按文件类型分类
```

常用变量：

| 变量 | 含义 |
|------|------|
| `[year]` | 文件修改年份 |
| `[month]` | 文件修改月份 |
| `[day]` | 文件修改日期 |
| `[extension]` | 文件扩展名 |
| `[filename]` | 原文件名（不含扩展名） |

## 典型使用场景

### 场景一：自动整理下载文件夹

**目标**：将下载文件夹中的文件按类型自动分类到不同子文件夹。

**规则配置示例**：

| 规则名 | 过滤器 | 动作 | 目标路径 |
|--------|--------|------|----------|
| 图片整理 | `*.jpg;*.png;*.webp;*.gif` | 移动 | `D:\Downloads\Images\` |
| 文档整理 | `*.pdf;*.docx;*.xlsx;*.pptx` | 移动 | `D:\Downloads\Documents\` |
| 视频整理 | `*.mp4;*.mkv;*.avi;*.mov` | 移动 | `D:\Downloads\Videos\` |
| 压缩包整理 | `*.zip;*.rar;*.7z` | 移动 | `D:\Downloads\Archives\` |
| 安装程序 | `*.exe;*.msi` | 移动 | `D:\Downloads\Setup\` |

**设置自动监控**（见后文"关联文件夹"章节），即可实现无需手动操作的全自动整理。

---

### 场景二：按日期归档照片

**目标**：将手机导入的照片按年份和月份归档。

**规则配置**：

- 过滤器：`*.jpg;*.jpeg;*.heic;*.png`
- 动作：移动
- 目标路径：`D:\Photos\[exif:year]\[exif:month]\`

DropIt 会读取照片的 EXIF 信息，自动按拍摄时间分类，例如将照片整理到 `D:\Photos\2025\08\` 目录中。

---

### 场景三：清理临时文件

**目标**：自动删除特定后缀的临时文件。

- 过滤器：`*.tmp;*.log;Thumbs.db;desktop.ini`
- 附加条件：修改日期早于 7 天
- 动作：删除

> ⚠️ **注意**：删除操作不可逆，建议先用"移动"到回收站文件夹，确认无误后再彻底删除。

---

## 使用配置文件管理不同场景

当你有多个不同用途的规则集时，可以用**配置文件（Profile）**来分组管理。

### 创建配置文件

1. 右键悬浮图标 → **"配置文件（Profiles）"**
2. 点击 **"新建"**，输入配置文件名称（如"工作文档"、"照片归档"）
3. 在该配置文件下添加专属规则

### 切换配置文件

右键悬浮图标，在配置文件列表中点击即可一键切换。不同场景使用不同配置文件，互不干扰。

---

## 关联文件夹（自动监控）

关联文件夹是 DropIt 最强大的功能之一，可让 DropIt **持续监控指定目录**，一旦有新文件出现，立即自动处理，完全不需要手动拖放。

### 设置步骤

1. 右键悬浮图标 → **"关联和规则"**
2. 点击左侧的 **"文件夹"** 标签页
3. 添加要监控的目录（如 `C:\Users\你的用户名\Downloads`）
4. 选择对应的配置文件
5. 设置**监控延迟**（建议 30 秒，防止文件未下载完就被处理）
6. 勾选 **"开机自动启动监控"**

完成后，DropIt 将在后台默默工作，自动整理所有新增文件。

## 常用技巧

### 技巧一：规则优先级

规则列表中，**越靠上的规则优先级越高**。如果一个文件匹配多条规则，只有最高优先级的规则会执行（除非开启"执行所有匹配规则"选项）。

建议将更具体的规则放在前面，通用规则放在末尾。

### 技巧二：测试规则

在正式使用前，可将动作设为 **"忽略（Ignore）"**，先观察哪些文件会被匹配，确认无误后再改为实际操作，避免误处理重要文件。

### 技巧三：导出与备份配置

整理好的规则和配置文件可以导出备份：

右键图标 → **"导入/导出（Import/Export）"** → 选择导出，保存为 `.dpi` 文件。

换电脑时直接导入，一键恢复所有配置。

### 技巧四：正则表达式匹配

如果通配符不够灵活，DropIt 还支持**正则表达式**过滤，适合处理复杂命名规则的文件：

```
^报告_\d{4}-\d{2}.*\.pdf$    # 匹配如 "报告_2026-03_季度总结.pdf"
```

## 小结

DropIt 凭借其简洁的拖放交互和强大的规则引擎，能够将日常繁琐的文件整理工作变为全自动流程。核心使用流程总结如下：

1. **定义规则** → 设定匹配条件和执行动作
2. **创建配置文件** → 按场景分组管理规则
3. **关联监控文件夹** → 实现全自动后台整理
4. **定期维护规则** → 根据实际需要调整和优化

合理利用 DropIt，可以彻底告别手动整理文件的烦恼，让你的磁盘始终保持井井有条。

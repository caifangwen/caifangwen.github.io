## 外贸的本质是什么

外贸的核心是**信息差套利 + 供应链整合**：你站在海外买家和中国工厂之间，用语言能力、选品眼光和履约能力赚取中间的价值。

独立站让你掌握了"卖"的能力——流量、转化、用户体验。但要真正跑通全流程，还需要补上"买"的能力，也就是**供应商对接和供应链管理**。

---

## 外贸全流程能力图谱

```
选品 → 找货 → 谈判 → 打样 → 下单 → 物流 → 履约 → 售后
  ↑                                                    ↓
  └──────────────── 数据反馈 & 选品迭代 ───────────────┘
```

### 1. 选品能力（你已有基础）

独立站经验让你最熟悉这一环。关键是把流量数据反向用于选品：

- 用 Google Trends、Semrush、亚马逊 BSR 验证需求
- 看独立站的搜索词报告，找高转化但供应不足的品类
- 判断「能卖」vs「能持续卖」：后者需要供应链稳定性

### 2. 找货能力（核心补课方向）

**线上渠道：**

- **1688.com**：最重要的国内采购平台，价格透明，支持小批量
- **阿里国际站（Alibaba）**：适合找有出口经验的工厂
- **全球速卖通 / 中国制造网**：补充渠道
- **Google 反查**：用产品关键词 + `site:made-in-china.com` 找工厂

**线下渠道（强烈推荐）：**

- **广交会**（每年4月、10月，广州）：综合展，适合建立初始供应商库
- **义乌小商品市场**：标品、礼品类首选
- **产业带直访**：深圳（电子）、东莞（家具）、佛山（陶瓷灯具）、宁波（小家电）

> **实操建议**：第一次找货，先在1688搜索，找3~5家报价，然后去实地验厂或参加展会交叉验证。

### 3. 供应商谈判能力

外贸谈判和国内采购的最大区别：**工厂更在乎长期关系，而不只是单笔订单价格**。

**谈判核心要素：**

| 要素 | 说明 |
|------|------|
| MOQ（最低起订量） | 新客户从小单切入，不要一上来就压价要大单 |
| 单价 | 量越大价越低，但要评估自己的资金周转 |
| 账期 | 惯例：T/T 30% 定金 + 70% 见提单；熟悉后可谈月结 |
| 交货期 | 明确生产周期，避免旺季延误 |
| 质检条款 | 提前约定验货标准，写入合同 |

**谈判话术原则：**
- 第一句话不谈价格，先问生产能力和月产量
- 给工厂「画饼」：说明你的销售量级和增长计划
- 让工厂觉得你是「值得培养的长期客户」

### 4. 打样与质检能力

这是新手最容易踩坑的环节。

**打样流程：**
1. 提供详细 spec sheet（规格书）：尺寸、材质、颜色、包装要求
2. 索取样品（付样品费 + 快递费，正规工厂都接受）
3. 收到样品后：外观 → 功能 → 包装 → 与描述一致性，逐项核查

**批量生产前的质检（QC）：**
- **出货前检验（Pre-shipment inspection）**：最重要，可自己去或委托第三方（如 SGS、Bureau Veritas）
- **AQL 抽样标准**：一般用 AQL 2.5，大货抽检比例约 10%~15%

### 5. 物流与清关能力

| 运输方式 | 适用场景 | 时效 | 成本 |
|----------|----------|------|------|
| 快递（DHL/FedEx/UPS） | 打样、小批量 | 3~7天 | 高 |
| 空运 | 急货、高价值轻货 | 5~10天 | 中高 |
| 海运 FCL（整柜） | 大批量 | 25~40天 | 低 |
| 海运 LCL（拼箱） | 中等批量 | 30~45天 | 中 |
| 专线物流 | 特定国家（如美国、欧洲） | 10~20天 | 中 |

**清关要点：**
- **HS 编码**：决定关税税率，务必准确
- **目的国认证**：美国（FCC/UL）、欧盟（CE）、德国（GS）——做独立站前必须确认是否需要
- **关税归谁承担**：DDP（卖家承担）vs DDU/DAP（买家承担）

### 6. 资金与外汇管理

- **收款工具**：Stripe、PayPal（独立站已有）；补充 Payoneer、PingPong、连连支付用于对接工厂付款
- **汇率风险**：美元结算为主，结汇时机影响利润，规模大了可考虑远期锁汇
- **账期管理**：工厂预付款 + 物流费 + 平台费用叠加，注意现金流节点

## 你的优先补课路径（基于独立站基础）

```
第1个月：熟悉1688 → 找3~5个潜在品类的供应商 → 打1~2款样品
第2个月：参加一次展会或产业带考察 → 敲定1家主力供应商
第3个月：小批量下单（50~200件）→ 走完完整物流流程
第4个月：根据实际销售数据复盘，优化选品和供应商结构
```

## 常见坑：新手外贸必知

1. **不签合同就打款**：无论多熟，一定要有书面合同或至少微信文字确认
2. **样品好但大货差**：打样和大货生产线不同，必须做出货前质检
3. **忽视认证问题**：欧美市场对产品认证要求严格，无认证可能被海关扣货或被投诉下架
4. **MOQ 压力下过度备货**：先跑通小单再放量，独立站流量有波动，不要被工厂「忽悠」压仓库
5. **只靠一家供应商**：主力供应商 + 备用供应商，避免被掐脖子

## 总结：你的核心竞争力组合

> 独立站技能（流量 + 转化 + 用户体验）  
> \+ 供应链能力（选品 × 找货 × 质检 × 物流）  
> = **真正的外贸闭环**

独立站是你的最大优势——很多传统外贸工厂老板恰恰缺这个。你补上供应链这条腿，就形成了别人难以复制的组合壁垒。

从**一款产品、一家工厂、一条物流线路**开始跑通，比什么都强。

红楼梦_人物关系设计文档

文章摘要

游戏开发

《红楼梦》经营策略竞技游戏 — 人物关系设计文档

一、世界观与游戏框架

游戏背景：贾府鼎盛至衰落的十年间。玩家选择一名角色，在荣宁二府的权力网络中生存、积累资源、实现私欲，最终在"抄家"结局中以个人评分决出胜负。

核心机制：

  • 每回合代表一个季节，共 40 回合（10年）
  • 角色间可结盟、交易、背刺、联姻、告密
  • 每个角色有隐藏目标（私欲）和公开阵营
  • 最终评分 = 生存分 + 私欲达成分 + 关系网络分 - 污点分

二、阵营体系

阵营代表颜色核心利益
🔴 荣国府嫡系朱红维护贾氏主脉权威与财产
🟡 宁国府支脉金黄扩张旁支势力，蚕食主脉
🟣 外戚联盟紫借贾府之势为娘家谋利
🔵 清流文人青蓝诗书传家，远离权斗
⚫ 暗流势力墨黑个人野心，两面下注
⚪ 边缘幸存者素白明哲保身，伺机而动

三、核心角色详解

🔴 贾母（史太君）

  • 阵营：荣国府嫡系 · 最高权威
  • 身份：荣国府老祖宗，贾政之母，贾赦之母
  • 核心资源：威望值（全场最高）、私库财富、裁决权
  • 私欲：
    • 主欲：维持贾府表面荣光，死前不见家族败落
    • 隐欲：偏爱宝玉，希望宝黛成婚（但会被现实妥协）
    • 执念：自己的体面与享乐不受侵犯
  • 特殊技能：
    • 「老祖宗裁决」：每季度可否决任意一项家族决策
    • 「溺爱光环」：宝玉在其庇护下免疫一次惩罚
    • 「压箱底」：拥有隐藏私库，可在危机时一次性释放大量资源
  • 弱点：年老体衰，健康值每5回合-1；对孙辈过分溺爱易被利用
  • 合作倾向：王夫人（表面）、凤姐（依赖）、宝玉（溺爱）
  • 竞争对象：邢夫人（不喜）、贾赦（嫌弃）
  • 评分标准：去世时贾府财产≥初始值60% (+50分)；宝玉婚事符合心意 (+30分)；享乐指数全程维持高位 (+20分)

🔴 王夫人

  • 阵营：荣国府嫡系 · 实权派
  • 身份：贾政正妻，宝玉之母，凤姐姑母
  • 核心资源：月例分配权、宗教影响力、对宝玉婚事的最终话语权
  • 私欲：
    • 主欲：为宝玉娶薛宝钗，巩固金玉良缘
    • 隐欲：清除一切她认为"妖媚"的威胁（晴雯、金钏儿等）
    • 深层恐惧：宝玉被林黛玉"带坏"，走上与贾珠同样的死路
  • 特殊技能：
    • 「月例裁夺」：可扣押/增加任意角色的月例银子
    • 「驱逐令」：可将丫鬟/清客逐出贾府（消除小角色）
    • 「吃斋念佛」：每季度可免疫一次道德污点惩罚
  • 弱点：实际管理能力弱，长期依赖凤姐；容易被宗教蒙蔽
  • 合作倾向：薛姨妈（亲密）、凤姐（利用）、宝钗（力推）
  • 竞争对象：林黛玉（威胁）、晴雯（厌恶）、赵姨娘（鄙视）
  • 评分标准：宝钗嫁宝玉 (+40分)；成功驱逐≥3名"威胁"人物 (+20分)；宝玉金榜题名 (+40分)

🔴 贾政

  • 阵营：荣国府嫡系 · 道德牌坊
  • 身份：荣国府二老爷，王夫人之夫，宝玉之父
  • 核心资源：官职影响力、对宝玉的惩戒权、清流声望
  • 私欲：
    • 主欲：宝玉走科举正途，光耀门楣
    • 隐欲：自己升迁，在朝廷站稳脚跟
    • 秘密：内心欣赏宝玉才华，但绝不承认
  • 特殊技能：
    • 「廷杖威慑」：可对宝玉施加"管教"，暂时提升宝玉学业值但降低其关系值
    • 「清流交际」：每季度可结交一名朝廷文官，获得政治资源
    • 「面子工程」：可消耗大量财富举办文会，大幅提升家族声望
  • 弱点：政治能力平庸，清客门客多为蠹虫；对内宅事务几乎无知
  • 合作倾向：清客群（利用）、王夫人（夫妻）
  • 竞争对象：贾赦（不合）、宝玉（父子矛盾）
  • 评分标准：升迁至三品以上 (+30分)；宝玉高中进士 (+50分)；家族声望维持 (+20分)

🔴 贾宝玉

  • 阵营：清流文人 / 荣国府嫡系（双重）
  • 身份：荣国府嫡长孙，衔玉而生
  • 核心资源：贾母溺爱庇护、大观园核心社交网络、玉石（神秘加成）
  • 私欲：
    • 主欲：与黛玉长相厮守，远离仕途经济
    • 隐欲：守护大观园所有姐妹不受伤害
    • 深层：对这个世界的本质怀疑，向往出世
  • 特殊技能：
    • 「通灵宝玉」：每季度可感知一个隐藏事件（情报能力）
    • 「众女心悦」：与女性角色建立关系的成本-50%
    • 「叛逆光环」：可拒绝任意一项指派任务而不受惩罚（每季度1次）
  • 弱点：无任何经济生产能力；被众多势力视为棋子；健康值随黛玉健康值联动
  • 合作倾向：林黛玉（灵魂伴侣）、史湘云（挚友）、晴雯（守护）
  • 竞争对象：薛宝钗（被安排的命运）、贾政（管束）
  • 评分标准：与黛玉关系值满格至结局 (+60分)；出家/出走（另类胜利，+100分但需满足条件）；大观园诗社存续 (+20分)

🟣 林黛玉

  • 阵营：外戚联盟（寄居）/ 清流文人
  • 身份：贾母外孙女，林如海之女，寄居荣国府
  • 核心资源：诗才（声望加成）、宝玉情感资本、贾母庇护
  • 私欲：
    • 主欲：与宝玉名正言顺结合
    • 隐欲：证明自己不是"寄人篱下"的可怜人，赢得真正的尊重
    • 深层恐惧：被抛弃，如父母一样早逝
  • 特殊技能：
    • 「诗词锋芒」：可用诗词攻击任意角色声望（舌战）
    • 「慧眼识人」：可识破任意角色一个隐藏私欲（侦查）
    • 「泪水攻势」：对宝玉使用时，宝玉必须优先响应黛玉需求
  • 弱点：体弱多病，健康值全场最低；无经济来源；情绪波动影响行动力
  • 合作倾向：宝玉（核心）、紫鹃（依赖）、史湘云（时好时坏）
  • 竞争对象：薛宝钗（直接情敌）、王夫人（结构性威胁）
  • 评分标准：宝玉明确拒绝宝钗 (+30分)；诗社声望全场最高 (+20分)；健康存活至第30回合 (+50分——极难）

🟣 薛宝钗

  • 阵营：外戚联盟（薛家）/ 暗流势力
  • 身份：薛姨妈之女，王夫人外甥女，金陵十二钗之首
  • 核心资源：薛家财富输血、金锁（命定加成）、八面玲珑社交网络
  • 私欲：
    • 主欲：嫁给宝玉，完成金玉良缘
    • 隐欲：重振薛家门楣，成为荣国府真正的主母
    • 深层：她渴望被真正欣赏，而非只是"符合标准"
  • 特殊技能：
    • 「滴翠亭手段」：可将自己的污点嫁祸给其他角色
    • 「冷香丸魅力」：每季度可将一名敌对角色转为中立
    • 「薛家钱袋」：每季度可为贾府注入一笔额外资金（换取好感）
  • 弱点：被众人视为"功利"，道德污点一旦暴露伤害极大；薛蟠随时可能惹祸
  • 合作倾向：王夫人（策略同盟）、薛姨妈（后盾）、香菱（表面关照）
  • 竞争对象：林黛玉（情敌）、探春（潜在政治竞争）
  • 评分标准：嫁宝玉 (+40分)；薛家未破产 (+20分)；成为荣国府管家 (+30分)；宝玉真心爱她 (+50分，极难）

🔴 王熙凤

  • 阵营：荣国府嫡系 / 暗流势力（实为双面）
  • 身份：贾琏之妻，王夫人侄女，荣国府实际管家
  • 核心资源：财务控制权、情报网络（心腹丫鬟）、执行力
  • 私欲：
    • 主欲：牢牢把持荣国府管家权，积累个人财富
    • 隐欲：清除尤二姐等威胁，让贾琏彻底臣服
    • 深层恐惧：失去权力后一无所有
  • 特殊技能：
    • 「协理宁国府」：可跨阵营干涉宁国府事务
    • 「放高利贷」：以贾府名义放贷，每季度获得额外收益（但风险积累）
    • 「借刀杀人」：可借王夫人/贾母之手打击对手
    • 「铁腕管家」：每季度可强制执行一项经济决策
  • 弱点：贾琏暗中积累反制资源；健康值中等；积累的黑料随时可被揭发
  • 合作倾向：贾母（效忠）、王夫人（利用）、平儿（依赖）
  • 竞争对象：尤二姐（宿敌）、探春（管理权竞争）、赵姨娘（互相憎恨）
  • 评分标准：掌握管家权≥25回合 (+40分)；个人积累财富≥3000两 (+30分)；贾琏未纳妾成功 (+20分)；安然度过抄家 (+50分，极难）

🟡 贾珍

  • 阵营：宁国府支脉 · 主脑
  • 身份：宁国府族长，贾蓉之父
  • 核心资源：宁国府产业、族长名义权力、秦可卿旧有关系网
  • 私欲：
    • 主欲：扩大宁国府影响力，压过荣国府
    • 隐欲：声色犬马，寻欢作乐不受约束
    • 秘密：与秦可卿的不伦关系，是最大隐患
  • 特殊技能：
    • 「族长令」：可召集贾氏族人执行一项集体决定
    • 「宁荣通道」：可自由调用两府部分共有资源
    • 「声色场所」：可消耗财富举办宴会，大幅提升所有参与者好感度
  • 弱点：道德污点极高；秦可卿事件是随时爆发的定时炸弹；贾蓉软弱
  • 合作倾向：贾蓉（父子）、薛蟠（酒肉朋友）、贾赦（同流合污）
  • 竞争对象：贾政（路线之争）、王熙凤（利益摩擦）
  • 评分标准：宁国府未被率先抄家 (+30分)；宴乐享受值最高 (+20分)；秘密未被揭发 (+40分）

🟡 秦可卿

  • 阵营：宁国府支脉（早期）/ 边缘幸存者
  • 身份：贾蓉之妻，宁国府少奶奶，身世成谜
  • 核心资源：众人缘（全场最广）、托梦预言能力、隐藏身世
  • 私欲：
    • 主欲：揭开自己的身世之谜，寻求真正的归属
    • 隐欲：用有限的生命为贾府留下一线生机（托梦王熙凤）
    • 深层痛苦：被贾珍纠缠，无法拒绝，寻求解脱
  • 特殊技能：
    • 「托梦警示」：死亡时可为一名玩家留下一个预言提示
    • 「万人迷」：与任意角色初始关系值+20
    • 「隐秘身世」：若身世被查明，可触发特殊剧情分支
  • 弱点：健康值极低（注定早逝）；被贾珍控制；无独立经济来源
  • 评分标准（特殊——早死角色）：死前完成托梦 (+50分)；身世被揭明 (+30分)；死后贾府按她建议保留家族资产 (+40分)

⚫ 贾赦

  • 阵营：暗流势力（荣国府长房）
  • 身份：荣国府长子，邢夫人之夫，贾琏之父
  • 核心资源：长房继承权（法理优势）、古玩字画收藏
  • 私欲：
    • 主欲：抢回荣国府主导权（目前被贾政把持）
    • 隐欲：鸳鸯，强行纳妾（具体执念）
    • 本质：贪婪、好色、懒惰，一切行为出于私欲
  • 特殊技能：
    • 「长房夺权」：可在关键时刻主张长房继承权，引发家族内乱
    • 「古玩勒索」：可强迫他人出售珍贵物品给自己
  • 弱点：贾母厌恶；与贾政长期不合；邢夫人无能
  • 合作倾向：贾珍（腐化同盟）、薛蟠（狐朋狗友）
  • 竞争对象：贾政（核心宿敌）、贾母（被压制）
  • 评分标准：夺得荣国府主导权 (+50分)；纳鸳鸯为妾 (+20分)；个人财产超过贾政 (+30分)

🟣 史湘云

  • 阵营：外戚联盟（史家）/ 清流文人
  • 身份：贾母侄孙女，史家千金，大观园常客
  • 核心资源：史家背景、豪爽人缘、诗才
  • 私欲：
    • 主欲：找到真心相待的知己（宝玉？卫若兰？）
    • 隐欲：摆脱史家对她的压榨（在家需做针线活）
    • 深层：渴望自由与认可，不愿被规则束缚
  • 特殊技能：
    • 「醉卧芍药」：可用豪爽之气化解任意一次场景紧张局势
    • 「史家外援」：危急时可召唤史家资源救急（每局1次）
  • 合作倾向：宝玉（挚友）、宝钗（崇拜）、黛玉（亦敌亦友）
  • 评分标准：与宝玉维持最高友谊值 (+20分)；嫁得良人 (+40分)；诗社活跃度最高 (+20分)

🔴 探春（贾探春）

  • 阵营：荣国府嫡系（庶出困境）/ 暗流势力
  • 身份：贾政庶女，赵姨娘之女，才干卓著
  • 核心资源：管理才能、改革方案、骨气与清醒
  • 私欲：
    • 主欲：用才能证明自己，突破庶出身份的天花板
    • 隐欲：推动贾府改革，延缓衰败（她看得最清楚）
    • 深层悲剧：她知道无论多努力，最终仍逃不过远嫁的命运
  • 特殊技能：
    • 「兴利除弊」：可推行一项改革，每回合增加贾府收益
    • 「秋爽斋令」：可强制拆散任意两个角色的合作关系
    • 「庶出之刃」：可揭发任意一名嫡出角色的隐秘污点（但自损声望）
  • 弱点：赵姨娘随时可能给她添乱；庶出身份是永久的政治弱点
  • 合作倾向：宝钗（志同道合）、李纨（表面合作）、平儿（信任）
  • 竞争对象：王熙凤（管家权）、赵姨娘（尴尬母女）
  • 评分标准：改革措施≥3项落实 (+30分)；管家权≥10回合 (+30分)；远嫁前完成家族重要使命 (+40分)

⚪ 李纨

  • 阵营：边缘幸存者
  • 身份：贾珠遗孀，贾兰之母，大观园稻香村主人
  • 核心资源：寡妇道德资本（免疫大量攻击）、贾兰培育潜力
  • 私欲：
    • 主欲：让贾兰出人头地，一雪贾珠早逝之耻
    • 隐欲：在乱局中保全自己和贾兰的财产
    • 表面：青灯古佛，实则精打细算
  • 特殊技能：
    • 「寡妇牌坊」：每季度免疫一次任意角色的资源掠夺
    • 「诗社盟主」：可召集所有诗社成员，获得集体声望加成
  • 评分标准：贾兰科举成功 (+60分)；个人财产在抄家后仍有剩余 (+40分)

⚫ 赵姨娘

  • 阵营：暗流势力（内宅底层）
  • 身份：贾政妾室，探春、贾环之母
  • 核心资源：几乎没有——只有阴谋能力
  • 私欲：
    • 主欲：让贾环继承家业，压倒宝玉
    • 隐欲：让王夫人、凤姐倒台，自己扬眉吐气
    • 本质：被压迫者的愤怒，以最笨拙的方式反抗
  • 特殊技能：
    • 「魇魔法」：可发动诅咒（随机事件），成功率低但伤害极高
    • 「内宅搅局」：每季度可制造一次家庭矛盾，降低贾政家庭和谐值
  • 弱点：几乎没有盟友；行事愚蠢，往往搬起石头砸自己的脚
  • 评分标准：贾环获得任何形式的继承权 (+40分)；凤姐或王夫人下台 (+30分)；存活至结局 (+30分，对她来说很难）

🔵 晴雯

  • 阵营：清流文人（大观园生态）
  • 身份：宝玉第一大丫鬟，“心比天高”
  • 核心资源：宝玉信任、刺绣技艺、反抗精神
  • 私欲：
    • 主欲：在规则最多的地方，活出最真实的自己
    • 隐欲：宝玉能给她一个名分（但又不愿承认）
    • 深层：恐惧被"好风凭借力"的人算计
  • 特殊技能：
    • 「勇晴雯病补雀金裘」：危机时可以自损健康换取一次极高成就
    • 「芙蓉花神」：死后可在关键时刻为宝玉传递一个信息
  • 弱点：健康值低；口舌伤人，树敌多；无任何政治靠山
  • 评分标准：未被逐出贾府 (+30分)；宝玉为其作《芙蓉女儿诔》 (+40分)；王善保家的阴谋被揭穿 (+30分)

⚪ 平儿

  • 阵营：边缘幸存者（最理性）
  • 身份：王熙凤陪嫁丫鬟，贾琏通房，凤姐心腹
  • 核心资源：两边信任（凤姐+贾琏）、情报整合能力、处事智慧
  • 私欲：
    • 主欲：在夹缝中保全自身，等待时局明朗
    • 隐欲：若凤姐倒台，能平稳过渡，不被牵连
    • 深层：渴望真正被尊重，而不只是工具
  • 特殊技能：
    • 「八面玲珑」：可同时维持两个对立角色对自己的好感
    • 「明哲保身」：当任意角色发动攻击时，可宣布中立免受波及
  • 评分标准：凤姐结局无论好坏，平儿均能独立生存 (+50分)；积累秘密情报≥5条 (+30分)

🟣 薛姨妈

  • 阵营：外戚联盟（薛家）
  • 身份：王夫人妹妹，薛宝钗、薛蟠之母
  • 核心资源：薛家财富（递减）、王家关系、温柔外交手腕
  • 私欲：
    • 主欲：宝钗嫁入贾府，薛家与贾府彻底绑定
    • 隐欲：压制薛蟠惹祸，维持薛家体面
    • 深层：对王夫人有心理优越感，但又需要依附她
  • 特殊技能：
    • 「慈母攻势」：可用温情外交将任意角色的敌意降低
    • 「薛家注资」：每季度可选择是否向贾府输血（换取政治筹码）
  • 评分标准：宝钗完婚 (+50分)；薛蟠未入狱 (+20分)；薛家财产剩余≥50% (+30分)

⚫ 贾琏

  • 阵营：暗流势力（荣国府长房）
  • 身份：贾赦之子，王熙凤之夫
  • 核心资源：外务交际、秘密积累的反制资源
  • 私欲：
    • 主欲：摆脱凤姐控制，纳尤二姐
    • 隐欲：继承荣国府，而非一直被凤姐架空
    • 深层：其实软弱，只想安逸享乐
  • 特殊技能：
    • 「外务斡旋」：可在朝廷/商界完成一次凤姐无法干涉的独立交易
    • 「秘密娶妾」：可在凤姐不知情下与外部角色建立关系
  • 竞争对象：王熙凤（婚姻权力斗争）
  • 评分标准：尤二姐存活并正式纳入家门 (+40分)；财产超过凤姐私藏 (+30分）

🔵 紫鹃

  • 阵营：清流文人（大观园生态）
  • 身份：林黛玉贴身丫鬟，实为贾母所赐
  • 私欲：
    • 主欲：帮助黛玉与宝玉终成眷属
    • 隐欲：在黛玉死后，仍保有尊严地活下去
  • 特殊技能：
    • 「试玉」：可制造宝黛关系危机以测试宝玉真心（高风险高收益）
    • 「护玉」：可每季度为黛玉抵挡一次健康损耗
  • 评分标准：黛玉与宝玉关系值≥80分存续 (+40分)；黛玉死后仍守护其遗愿 (+30分)

Claude 电脑使用功能正式推出：让 AI 替你操控电脑

Anthropic 旗下 AI 助手 Claude 正式推出电脑使用功能，支持在 Claude Cowork 和 Claude Code 中研究预览，目前仅限 macOS 平台。

AI工作流 #Claude #AI #Anthropic #Computer Use #Claude Code #自动化

Claude 现在可以帮你操控电脑了

Anthropic 官方账号 @claudeai 近日在 X（原 Twitter）上宣布：你现在可以允许 Claude 使用你的电脑来完成各种任务。

这项功能标志着 AI 助手从"对话工具"正式迈入"自主执行"时代。

它能做什么？

Claude 的电脑使用功能涵盖了几乎所有你在桌面前会做的事情：

  • 打开应用程序 —— 自动启动你需要的软件
  • 浏览器操控 —— 自主导航网页、填写表单
  • 电子表格操作 —— 读取、填写、整理数据
  • 任何桌面任务 —— 只要你能在桌前完成，Claude 就能替你做

官方给出了一个生动的使用场景示例：

“我要迟到了，你能帮我把 pitch deck 导出为 PDF，并附在我下午 2 点的日历邀请里吗？”

这正是该功能的核心价值——当你分身乏术时，让 Claude 替你处理繁琐的电脑操作。

如何体验？

目前该功能以**研究预览（Research Preview）**形式开放，支持以下两个平台：

平台 说明
Claude Cowork Anthropic 推出的桌面协作工具
Claude Code 面向开发者的命令行 AI 编程助手

⚠️ 当前仅支持 macOS，Windows 和 Linux 支持尚未公布时间表。

为什么这很重要？

过去的 AI 助手只能"说"，现在的 Claude 开始"做"了。这种从语言模型到**行动代理（Agentic AI）**的转变，意味着：

  1. 效率革命：重复性桌面操作可以完全委托给 AI
  2. 无缝集成：AI 不再局限于对话框，而是融入真实工作流
  3. 自然语言指令：不需要写脚本或配置自动化，一句话即可驱动复杂操作

小结

Claude 的电脑使用功能是 Anthropic 在 Agentic AI 赛道上的重要布局。随着这一能力逐步开放和完善，AI 助手正在从"聊天机器人"演变为真正意义上的数字员工。

如果你是 macOS 用户，不妨前往 claude.ai 了解 Claude Cowork 或 Claude Code，率先体验这一前沿功能。

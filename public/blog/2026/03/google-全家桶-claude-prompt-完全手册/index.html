💡 使用指南

Prompt 使用原则

原则 说明
确认后执行 涉及发送、创建、修改的操作，加上"请先让我确认"
明确范围 搜索类 Prompt 注明时间范围、关键词
分步执行 复杂工作流拆分为多个 Prompt 依次执行
保留草稿 邮件/文档类优先生成草稿而非直接提交

移动端使用技巧

  • 使用短句 Prompt，减少输入量
  • 利用"告诉我两行字"控制输出长度
  • 语音输入 → Claude 润色 → 发送，减少打字

当前已连接服务

  • ✅ Gmail（已连接）
  • ✅ Google Calendar（已连接）
  • ⬜ Google Drive（需在设置中开启）
  • ⬜ Google Tasks / Docs / Sheets / Slides（需扩展 MCP 配置）
